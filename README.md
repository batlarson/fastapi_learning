# FastAPI Learning

Proyecto de aprendizaje de FastAPI — API REST básica con CRUD, validaciones con Pydantic y manejo de errores.

## Stack
- FastAPI
- Pydantic
- Uvicorn

## Ejecutar
```bash
uvicorn main:app --reload
```

## Endpoints
- GET /activos
- POST /activos
- PUT /activos/{ticker}
- DELETE /activos/{ticker}
